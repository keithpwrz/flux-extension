## Short Description (132 chars max)
Find the best Roblox servers by region. Live player counts, ping ratings, and one-click join — free, no account needed.

## Full Description
Flux helps Roblox players find the best servers to reduce lag and improve ping. Tired of high ping, lag spikes, or unstable connections when playing Roblox games? Flux shows you exactly where each server is located — so you can pick the one closest to you for the smoothest Roblox gameplay.

**Why Players Choose Flux Over Other Server Finders:**
- 🚀 **One-click scanning** — Open any Roblox game page and Flux instantly shows available servers
- 🌍 **16 datacenter locations** — Frankfurt, Paris, Amsterdam, London, Singapore, Tokyo, Mumbai, Los Angeles, Ashburn, Chicago, Dallas, Miami, New York, Seattle, Sydney, São Paulo
- 📊 **Live player counts** — See exactly how many players are in each server, refreshed every 5 seconds
- 🟢 **Color-coded ping ratings** — Green = best connection, Yellow = okay, Red = avoid. No guesswork.
- 🎯 **Direct server join** — Click "Join" and Roblox launches straight into the exact server you picked
- 🎨 **Clean dark dashboard** — Blends into Roblox's native design, doesn't feel like a third-party tool
- 🔒 **100% private** — No accounts, no signup, no data collection. Everything runs locally in your browser.
- 💰 **Completely free** — No subscriptions, no in-app purchases, no "premium" lock. All features are free.

**How It Works:**
1. Open any Roblox game page on roblox.com
2. Click the blue Flux button (with the "F" icon) next to the Play button
3. Browse servers organized by geographic region
4. See live player counts, player avatars, and color-coded ping ratings
5. Click "Join" on the best server — Roblox launches directly into it

**Perfect For:**
- Players tired of being auto-assigned to laggy servers
- Trading games where server location matters
- Competitive games where ping impacts gameplay
- Friends trying to join the same server in different regions

**Privacy:**
Flux does not collect, store, or transmit any personal data. All server data is fetched directly from Roblox's public API through your browser. No external servers. No analytics. No tracking. See our full privacy policy on GitHub.
