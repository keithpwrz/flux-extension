## Short Description (132 chars max)
Find Roblox servers by region. See live player counts, ping ratings, and join the best server in one click.

## Full Description
Flux lets you browse Roblox game servers by geographic region, so you always join the best server for your connection.

**Features:**
- One-click server scanning on any Roblox game page
- 16 datacenter locations worldwide — Frankfurt, Paris, Amsterdam, London, Singapore, Tokyo, Mumbai, Los Angeles, Ashburn, Chicago, Dallas, Miami, New York, Seattle, Sydney, São Paulo
- Live player counts and avatars per server — refreshed every 5 seconds
- Color-coded ping ratings (green/yellow/red) so you know which servers are fastest
- Direct server join — launches Roblox into the exact server you pick
- Clean dark dashboard that blends into Roblox's design
- No accounts, no subscriptions, no data collection

**How It Works:**
1. Open any Roblox game page
2. Click the Flux button (blue, with the "F" icon) beside the Play button
3. Pick a region from the sidebar
4. Browse servers with real player counts, avatars, and ping
5. Click "Join" on the server you want — Roblox launches directly into it

**Privacy:**
Flux does not collect, store, or transmit any personal data. All server data is fetched directly from Roblox's public API through your browser. No external servers, no analytics, no tracking. See our full privacy policy on GitHub.
